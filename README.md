# Florida Real Estate Statistical Analysis Project

## Project scope
This project performs an end-to-end statistical analysis on Florida real estate sales data. It includes:
- `analysis.ipynb` (data loading, cleaning, descriptive stats, visualization, hypothesis testing)
- `Statistical_Analysis_Report.md` (report narrative, interpretation, limitations, references)
- `requirements.txt` (pinned dependencies for reproducibility)
- `florida_real_estate_sold_properties_ultimate.csv` (source dataset)

## Reproducibility
1. Clone the Git repository from the `master` branch, as it contains the production-ready codebase. The `development` branch is available for continuous improvements and future updates.
   - `git clone -b master https://github.com/atamayop2018/ai-programming-foundations-project2`
2. Install dependencies using:
   - `pip install requirements.txt`
3. Run the Jupyter notebook:
   - `jupyter notebook analysis.ipynb`
4. Execute all cells in order.

## Analysis checklist
- [x] Dataset loads correctly and is inspected.
- [x] Descriptive statistics are computed and explained.
- [x] ≥3 visualizations with titles and axis labels.
- [x] Hypothesis tests include null/alternative statements and p-values.
- [x] Reproducibility notes and version tags are captured.
- [x] Report includes required sections and citations.

## Citation requirement
- Reproducibility reference: Peng (2011) and at least one additional peer-reviewed source.
- Methodological references: Cumming (2014) plus general statistical textbooks.
